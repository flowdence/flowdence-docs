## When to Use This

Use this procedure when you need to remove MuleSight from a space/site and avoid confusion for existing users.

## Before You Uninstall

1. Notify users that dashboard refresh, macro refresh, and ops publishing will stop.
2. Capture key screenshots/exports if you need audit evidence.
3. Decide whether to keep existing MuleSight-generated Confluence pages as historical records.

## Uninstall Steps

1. Open Confluence app management and uninstall MuleSight.
2. Return to the space and verify MuleSight pages/macros are no longer active.
3. Archive or keep historical ops pages based on your retention policy.

## Expected Effects

- MuleSight dashboard routes stop rendering app data.
- Macro refresh actions stop returning fresh snapshots.
- Scheduled drift incident/resolution publishing stops.
