---
title: "MuleSight Overview"
---

MuleSight brings MuleSoft operational visibility directly into Confluence. It is designed so teams can monitor drift, inspect API security posture, and embed live snapshot macros in documentation pages.

## Who This Is For

- Confluence space admins setting up MuleSight for a team space.
- Platform and integration engineers who need environment comparison and security posture checks.
- Documentation owners who want live MuleSoft snapshots in runbooks and architecture pages.

## What MuleSight Does

- Stores MuleSoft credentials per Confluence space.
- Compares `API Manager` and `CloudHub` resources across environments.
- Flags drift in resource presence, version, and status.
- Shows per-API security details (policies, tiers, contracts).
- Publishes ops incident/resolution history pages from scheduled sync behavior.
- Converts MuleSoft links into rich macros on Confluence pages.
- Provides a space-context-aware Rovo guidance surface.
