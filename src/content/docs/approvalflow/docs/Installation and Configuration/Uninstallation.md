---
title: "Uninstallation"
---

## When to Uninstall

- Environment decommission.
- App replacement.
- Temporary removal during architecture changes.

## Uninstall Steps

1. Log in as Confluence/site admin.
2. Go to `Apps` -> `Manage apps`.
3. Locate Approval Flow.
4. Select `Uninstall`.
5. Confirm removal.

## Impact Notes

- Existing page content remains.
- Approval Flow UI elements (byline/status actions) disappear.
- Historical approval comments already posted on pages remain as content history.
- Reinstall may require revalidation of workflows and assignments.

## Recommended Pre-Uninstall Checklist

- Export key workflow names and approvers.
- Capture current queue/analytics snapshots.
- Notify affected authors/approvers.
