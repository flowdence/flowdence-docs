## Purpose

`Workflow Analytics` summarizes approval performance across a selected time period and workflow filter.

## What You Can Analyze

- Total submissions.
- In Approval count.
- Approved count.
- Changes Requested count.
- Cancelled count.
- Status distribution chart.

![Analytics tab](../assets/screenshots/admin-workflow-management/13-workflow-analytics-tab.png)

![Analytics with actual records](../assets/screenshots/approval-lifecycle/15-workflow-analytics-with-existing-page-records.png)

## Usage Pattern

1. Set start/end date.
2. Filter by workflow (or all workflows).
3. Click `Apply`.
4. Use cards/charts for reporting and operational review.
