## Purpose

The page byline is where authors and approvers interact with approval lifecycle state.

## Author Flow

1. Open an existing Confluence page.
2. Open byline status (`Draft`, `In Approval`, etc.).
3. Select workflow.
4. Click `Request approval` (if available).
5. Confirm status changes to `In Approval`.

![Author selects workflow and submits](../assets/screenshots/approval-lifecycle/02-author-selects-workflow-and-submits-existing-page.png)

![Author sees In Approval](../assets/screenshots/approval-lifecycle/03-author-sees-in-approval-status-existing-page.png)

## Approver Flow

1. Open the same page as approver.
2. Open byline panel.
3. Click `Approve` or `Reject`.
4. Add decision comment.
