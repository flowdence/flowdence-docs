## Purpose

`Approval Queue` gives a live, filterable list of approval items in the space.

## What You See

- Count cards (pending/approved/rejected).
- Table with page, workflow, submitter, date, and status.
- Status filter control.

![Approval Queue tab](../assets/screenshots/admin-workflow-management/12-approval-queue-tab.png)

![Approval Queue with real page records](../assets/screenshots/approval-lifecycle/14-approval-queue-with-existing-page-records.png)

## Typical Workflow for Admins

1. Open `Approval Queue` tab.
2. Filter by `Pending` to focus on in-progress work.
3. Open target page from queue row.
4. Review lifecycle comments/byline.
