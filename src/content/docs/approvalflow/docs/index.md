This documentation set covers the Approval Flow Confluence app from install through day-to-day operations, using real SPARROW/SPARROWSALES space data and Playwright-captured evidence.

## What This App Does

Approval Flow adds a governed approval lifecycle on Confluence pages:

- Create single-step or multi-step workflows with named approvers.
- Submit pages for approval from the page byline.
- Approve or reject with comments.
- Auto-post approval lifecycle comments on the page.
- Track approvals in Approval Queue and Workflow Analytics.
- Manage page/workflow assignments from Manage Space.
- Import workflows across spaces.

## Primary Roles

- `Space Admin`:
  - Creates workflows.
  - Assigns workflows.
  - Imports workflows from other spaces.
  - Monitors queue and analytics.
- `Author`:
  - Requests approval from byline.
  - Reviews status and comments.
- `Approver`:
  - Reviews page.
  - Approves or rejects with comments.

## Quick Start Path

If you are new to the app, follow this order:

1. Install and verify app visibility.
2. Configure at least one workflow.
3. Run first approval cycle on an existing page.
4. Review queue and analytics.
5. Review comments/audit behavior.
