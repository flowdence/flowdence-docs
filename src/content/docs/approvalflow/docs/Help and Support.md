## Primary Troubleshooting

## 1. Approval byline not visible on a page

Checks:

- Confirm app is installed in target site.
- Confirm user has access to space/page.
- Confirm at least one active workflow exists in space.
- Refresh page after workflow updates.

## 2. Workflow not appearing in byline dropdown

Checks:

- Verify workflow is enabled in `Workflows` tab.
- Verify space context is correct.
- Confirm workflow import/creation completed successfully.

## 3. Approval Queue appears empty

Checks:

- Verify pending submissions exist for selected status filter.
- Clear filters/reset status selector.
- Confirm pages were actually submitted for approval.

## 4. Analytics cards show zero values

Checks:

- Adjust start/end date range.
- Verify workflow filter is correct.
- Click `Apply` after filter updates.

## 5. Manage Space rows/actions unavailable

Checks:

- Wait for page tree loading to complete.
- Expand parent nodes to reveal child pages.
- Confirm user has admin/owner permissions for assignment actions.

## Operational Escalation Data to Capture

Before escalation, capture:

- Space key and page URL.
- Workflow name.
- User role (author/approver/admin).
- Current status label shown in byline.
- Screenshot + video evidence from this docs package.
